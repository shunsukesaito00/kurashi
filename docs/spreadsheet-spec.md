# スプレッドシート仕様: 手取り試算シート

BOOTH「スタンダード版」向け。計算ロジックは [`tools/tedori.html`](../tools/tedori.html) の `computeNet()` と同一（2026年7月時点）。

**免責**: 概算です。健保組合・扶養・各種控除・住民税の実際の天引き開始時期は反映しません。正確な金額は給与明細・勤務先で確認してください。

## 前提・定数

| 名称 | 値 | 備考 |
|---|---:|---|
| 健康保険料率（本人） | 4.95% | 協会けんぽ目安。月収×率 |
| 厚生年金料率（本人） | 9.15% | 月収×率 |
| 雇用保険料率（本人） | 0.55% | 月収×率 |
| 基礎控除（所得税） | 480,000円/年 | |
| 基礎控除（住民税） | 430,000円/年 | |
| 住民税率 | 10% | 年額課税標準に適用 |
| 住民税均等割 | 5,000円/年 | 月割で ÷12 |
| 復興特別所得税 | ×1.021 | 所得税額に乗算 |

ボーナスにも社保合計率 `0.0495+0.0915+0.0055 = 0.1465` を適用する（`tedori.html` 同様）。

## 入力セル（手取り試算シート）

| セル | 名称 | 例 |
|---|---|---:|
| `B3` | 月収（額面・円） | 300000 |
| `B4` | 年間ボーナス（円） | 0 |

比較シート用にパターンA〜Cを並べる場合は `B3:D3`（月収）、`B4:D4`（ボーナス）。

## 中間計算（1パターン分・列 B の例）

セル参照は横にコピー可能（C列・D列は `$` を調整）。

### 1. 年間額面

```
年間額面  B5  = B3*12 + B4
```

### 2. 月額社会保険料

```
健康保険  B6  = ROUND(B3*0.0495, 0)
厚生年金  B7  = ROUND(B3*0.0915, 0)
雇用保険  B8  = ROUND(B3*0.0055, 0)
```

### 3. 年間社会保険料

```
年間社保  B9  = (B6+B7+B8)*12 + B4*0.1465
```

### 4. 給与所得控除（年額）

`tedori.html` の `salaryDeduction(gross)` と同じ分段関数。

**Google スプレッドシート（`B10`）:**

```
=IFS(
  B5<=1625000, 550000,
  B5<=1800000, B5*0.4-100000,
  B5<=3600000, B5*0.3+80000,
  B5<=6600000, B5*0.2+440000,
  B5<=8500000, B5*0.1+1100000,
  TRUE, 1950000
)
```

**Excel（`B10`）:**

```
=IF(B5<=1625000,550000,
 IF(B5<=1800000,B5*0.4-100000,
 IF(B5<=3600000,B5*0.3+80000,
 IF(B5<=6600000,B5*0.2+440000,
 IF(B5<=8500000,B5*0.1+1100000,
 1950000)))))
```

### 5. 課税所得（所得税用・年額）

```
B11  = MAX(0, B5 - B10 - B9 - 480000)
```

### 6. 所得税（年額・復興含む）

累進税率は `incomeTaxAnnual()` と同じ7段。

**Google スプレッドシート（`B12`）:**

```
=IF(B11<=0, 0,
 IFS(
  B11<=1950000, B11*0.05,
  B11<=3300000, B11*0.1-97500,
  B11<=6950000, B11*0.2-427500,
  B11<=9000000, B11*0.23-636000,
  B11<=18000000, B11*0.33-1536000,
  B11<=40000000, B11*0.4-2796000,
  TRUE, B11*0.45-4796000
 ) * 1.021
)
```

**Excel（`B12`）:** 上記と同様に `IF` をネスト（最大7段）。

### 7. 住民税（年額）

```
課税標準  B13  = MAX(0, B5 - B10 - B9 - 430000)
住民税年額 B14 = B13*0.10 + 5000
```

### 8. 月割り控除

所得税は**月給分のみ**月割り（ボーナス分を除外）。`tedori.html` と同じ比率。

```
月割所得税 B15 = IF(B5=0, 0, ROUND(B12/12 * (B3*12/B5), 0))
月割住民税 B16 = ROUND(B14/12, 0)
```

### 9. 毎月の手取り（結果）

```
手取り  B17 = B3 - B6 - B7 - B8 - B15 - B16
```

表示は `ROUND(B17, 0)`。サイトは `Math.round` で整数円。

### 10. 派生指標（任意）

```
控除合計（月）  = B3 - B17
手取り率        = B17 / B3
年間手取り目安  = B17 * 12   ※ボーナス月の変動は簡略化
```

## 手取り比較シート（3パターン）

| 行 | A列ラベル | B=パターンA | C=パターンB | D=パターンC |
|---|---|---|---|---|
| 3 | 月収（額面） | 入力 | 入力 | 入力 |
| 4 | 年間ボーナス | 入力 | 入力 | 入力 |
| 17 | 毎月の手取り | 式 | 式 | 式 |
| 18 | 手取り率 | `=B17/B3` | … | … |
| 19 | Aとの差（月） | — | `=C17-$B$17` | `=D17-$B$17` |
| 20 | Aとの差（年） | — | `=C19*12` | `=D19*12` |

サイトの比較表（[`tedori.html` 共有URL `?a=&b=&c=`](../tools/tedori.html)）と同じ見え方にできる。

## スプレッドシート実装メモ

- **丸め**: サイトは各段で `Math.round` 相当。式の最終手取りのみ `ROUND` でも誤差は±数円程度。厳密一致させる場合は社保・税の各段を `ROUND` する。
- **名前付き範囲**: 率を `健保率=0.0495` 等で定義すると改定時に1箇所修正で済む。
- **改定**: 社保料率は年1回、住民税・給与所得控除は税制改正時に要更新。毎年8月は育休上限とは別に手取り料率も要確認（[README](../README.md) 免責参照）。
- **新卒1年目**: 住民税天引きなしは未対応。必要なら `住民税免除フラグ` 列を追加し `B16=0` にする拡張余地あり。
