## 商品説明（本文・そのまま貼り付け可）

```
■ こんな方におすすめ

・転職・昇給・内定複数社の「手取り差」を数字で比較したい
・額面は上がるが、家計的に本当にプラスか確認したい
・ブラウザの計算結果を保存して、何度もシナリオを変えたい
・固定費・変動費を入れて「自由に使えるお金」「貯蓄率」を把握したい

■ この商品でできること

無料Webツール「くらしの計算室」の手取り計算をベースに、Excelで保存・編集できるテンプレートです。

【1】手取り試算 … 月収・ボーナスを入れると、社会保険料・所得税・住民税を概算した手取りが出ます
【2】手取り比較 … 最大3パターン（転職先A/B/Cなど）を横並びで比較。月・年の差額も表示
【3】家計サマリ … 手取り − 固定費 − 変動費 ＝ 自由に使える額。貯蓄率も確認
【4】積立メモ … 余ったお金を毎月積み立てた場合の将来額を簡易シミュレーション
【5】年間俯瞰 … 12か月分の手取り・支出・貯蓄を一覧（印刷・パートナーとの相談用）
【6】README … 使い方・免責事項・入力例

例）月収30万円・ボーナス0円 → 手取りの目安 約237,184円/月（概算）

■ 無料ツールとの違い

くらしの計算室（Web）: その場で計算・URL共有
本商品（Excel）: ファイルとして保存、家計項目の追加入力、印刷・オフライン編集

無料ツール: https://shunsukesaito00.github.io/kurashi/tools/tedori.html

■ 納品物

ダウンロード後、ZIPを解凍してください。

・tedori-kakei-template.xlsx（メインの Excel テンプレート・6シート）
・manual.pdf（使い方マニュアル 2ページ）

■ 動作環境

・Microsoft Excel、または Excel 形式（.xlsx）を開けるソフト（LibreOffice 等）
・編集・計算式の利用は PC 推奨（スマホは閲覧のみ想定）
・マクロなし・外部通信なし（オフラインで利用可能）

■ ご購入前に必ずお読みください（重要）

・本商品の手取り・税・社会保険料はすべて「概算」です
・健保組合の違い、扶養、各種控除、住民税の天引き開始時期など個別事情は反映していません
・給与明細・税務署・専門家の判断に代わるものではありません
・制度改正・料率改定時は、テンプレート内の数値・式の見直しが必要になる場合があります
・数字の最終確認は必ずご自身の給与明細等で行ってください

■ サポート

・使い方は同梱の manual.pdf および xlsx 内 README シートをご覧ください
・個別の税務相談・カスタマイズ対応は行っておりません
・不具合（ファイルが開けない等）のみ、BOOTHのメッセージ機能でご連絡ください

■ 更新について

2026年7月時点の料率・税制を前提に作成しています。大きな改正があった場合は内容の更新を検討します。
```
