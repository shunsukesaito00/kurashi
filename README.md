# くらしの計算室

日常でよく検索される計算(手取り・消費税・和暦変換など)をブラウザだけで完結させる、
日本語の無料ツールサイトです。Google AdSense による広告収益化を前提に設計しています。

## サイト構成

| ページ | 内容 |
|---|---|
| `index.html` | トップ(ツール一覧) |
| `tools/tedori.html` | 手取り計算(月収→手取り) |
| `tools/zeikomi.html` | 消費税計算(税込⇔税抜) |
| `tools/wareki.html` | 西暦・和暦変換 |
| `tools/nenrei.html` | 年齢計算 |
| `tools/bmi.html` | BMI・適正体重 |
| `tools/tsumitate.html` | 積立・複利シミュレーション |
| `tools/waribiki.html` | 割引(〇%オフ)計算 |
| `tools/jikyu.html` | 時給換算 |
| `tools/moji.html` | 文字数カウント |
| `privacy.html` | プライバシーポリシー(AdSense審査に必須) |
| `about.html` | 運営者情報(AdSense審査に有利) |

フレームワーク・ビルド不要の静的HTML/CSS/JSのみ。サーバーサイド処理はありません。

## ローカルでの確認

```bash
python3 -m http.server 8000
# http://localhost:8000 を開く
```

## 公開までの手順(必須)

1. **ホスティング**: GitHub Pages / Cloudflare Pages / Netlify のいずれかに無料デプロイ。
2. **独自ドメイン取得(強く推奨)**: AdSense はサブドメイン(`*.github.io` 等)では
   審査に通りにくいため、独自ドメイン(年1,000〜1,500円程度)を取得して割り当てる。
3. **プレースホルダの置換**:
   - `about.html` の運営者名・連絡先を実際の情報に書き換える
   - `sitemap.xml` と `robots.txt` の `YOUR-DOMAIN.example` を実ドメインに置換する
4. **Google Search Console に登録**し、`sitemap.xml` を送信する。

## 収益化ロードマップ(月5万円へ)

**重要な前提**: コードだけでは収益は発生しません。収益 = トラフィック × 収益化単価 であり、
トラフィックの獲得(SEO)には通常3〜12か月かかります。

### フェーズ1: AdSense 審査(公開後 2〜6週間)

- 公開後、[Google AdSense](https://www.google.com/adsense/) に申請する。
- 審査通過後、各ページの `.ad-slot`(広告スペースのプレースホルダ)を
  AdSense の広告コードに差し替える。
- 審査に落ちた場合はコンテンツ(解説文)を増やして再申請する。

### フェーズ2: 検索流入の育成(1〜6か月)

- 各ツールは「手取り 計算」「消費税 計算」など検索ボリュームの大きい
  キーワードを狙った構成になっている。競合が強いため、
  「手取り 計算 25万」「令和8年 西暦」のようなロングテール記事を
  ブログ形式で追加していくのが現実的。
- 目安: AdSense のページRPM(1,000PV当たり収益)は日本の一般サイトで
  200〜500円程度。月5万円には **月10〜25万PV** が必要。

### フェーズ3: 収益源の多様化(3か月目以降)

AdSense 単体で月5万は PV 要求が高いため、単価の高い収益源を併用する:

- **アフィリエイト(ASP)**: 積立ページ→証券口座開設(1件5,000〜15,000円)、
  時給・手取りページ→転職サービス(1件3,000〜15,000円)。
  A8.net・もしもアフィリエイト等に登録して広告を設置する。
  成果報酬型なら月5万は 5〜10件の成約で到達可能で、AdSense より現実的。
- **ツールの追加**: 検索需要のあるツール(例: 育休給付金計算、
  退職金税金計算、カロリー計算)を月2〜4本追加して入口を増やす。

## 免責

各ツールの計算は概算です。税率・保険料率は改正されるため、
`tools/tedori.html` の料率(協会けんぽ・厚生年金・雇用保険)は年1回見直してください。
